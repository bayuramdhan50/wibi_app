INSTRUKSI PENTING: Tambahkan file gambar berikut ke folder assets/images/

1. profile.png atau profile.jpg
   - Foto Anda untuk ditampilkan di Splash Screen dan halaman Biodata
   - Ukuran direkomendasikan: 500x500 pixels atau lebih

2. contact_1.png sampai contact_15.png
   - Foto untuk masing-masing kontak (15 kontak)
   - Ukuran direkomendasikan: 200x200 pixels
   - Atau gunakan placeholder avatar jika tidak ada foto

3. news_1.png sampai news_10.png
   - Gambar untuk berita (10 berita)
   - Ukuran direkomendasikan: 800x400 pixels (landscape)

Jika Anda tidak menambahkan gambar, aplikasi akan menggunakan icon placeholder secara otomatis.

Cara menambahkan gambar:
1. Buka folder: e:\Bayu\Koding\Joki\Mobile\wibi_app\assets\images\
2. Copy paste file gambar Anda ke folder tersebut
3. Pastikan nama file sesuai dengan yang tercantum di atas
4. Jalankan command: flutter pub get
5. Restart aplikasi
